
##Inital Setup
Install `Node.js` from [here]{https://nodejs.org).

Open a command prompt and change to the folder containing this `README.md` file.

Run `npm update`


##After getting latest code
Open a command prompt and change to the folder containing this `README.md` file.

Run `npm update`


##Running Server from cmd prompt
Open a command prompt and change to the folder containing this `README.md` file..

run `node server.js` to start server. Additional params list in `Startup Params` section.

by default the server runs on port 8080, to use a diffrent port use the `port` parameter.


###Startup Params
`port` = port to listen on. Default port 80. EX: `port=54321`
